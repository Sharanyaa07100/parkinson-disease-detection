# Parkinson Disease Detection Using Deep Neural Networks

This project detects Parkinson's Disease from speech/audio features using machine learning and deep learning models.

## Features
- Speech feature extraction using Praat-Parselmouth
- Jitter, Shimmer, and HNR feature extraction
- Data preprocessing using MinMaxScaler
- Model training using SVM, XGBoost, MLP, and CNN
- Performance metrics: Accuracy, Precision, Recall, F1-score
- Confusion matrix visualization
- Tkinter GUI for dataset upload, model training, prediction, and comparison graph

## Technologies Used
- Python
- Tkinter
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Keras
- TensorFlow
- Matplotlib
